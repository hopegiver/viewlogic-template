Automatic Data Fetching with dataURL

Eliminate manual API calls in component lifecycle hooks. Just define dataURL and data is automatically fetched!

Single API (Simple Usage)

export default {
    name: 'ProductsList',
    dataURL: '/api/products',  // ‚ú® Magic happens here!
    mounted() {
        // Data is already fetched and available!
        console.log('Products loaded:', this.products);
    }
};

Multiple APIs (Revolutionary!)

export default {
    name: 'Dashboard',
    dataURL: {
        products: '/api/products',
        categories: '/api/categories',
        stats: '/api/dashboard/stats'
    },  // ‚ú® Multiple APIs with named data!
    mounted() {
        // All APIs called in parallel, data available by name!
        console.log('Products:', this.products);
        console.log('Categories:', this.categories);
        console.log('Stats:', this.stats);
    }
};

Automatic Form Handling with Variable Parameters

Revolutionary form handling that eliminates manual submission logic. Use template syntax for dynamic parameters!

<!-- Dynamic form actions with variable parameters -->
<form action="/api/users/{userId}/posts" method="POST"
      data-success="handleSuccess">
    <input type="text" name="title" required>
    <textarea name="content" required></textarea>
    <button type="submit">Create Post</button>
</form>

<!-- Component logic - parameters resolved automatically -->
export default {
    data() {
        return {
            userId: 123    // {userId} will be replaced with this value
        };
    },
    methods: {
        handleSuccess(response) {
            console.log('Post created!', response);
        }
    }
};

DynamicInclude & HtmlInclude Components

Load dynamic content from any URL with full Vue reactivity and parameter injection.

<!-- DynamicInclude - Load ViewLogic pages dynamically -->
<DynamicInclude
    page="login"
    :use-cache="false"
    loading-text="Loading login page..."
    :params="{
        returnUrl: '/dashboard',
        theme: 'compact'
    }"
/>

<!-- HtmlInclude - Include raw HTML with XSS protection -->
<HtmlInclude
    src="/widgets/weather.html"
    :sanitize="true"
    loading-text="Loading widget..."
/>

üöÄ Quick Start

Get started with ViewLogic Router in minutes

<!-- Load ViewLogic Router -->
<script src="https://cdn.jsdelivr.net/npm/vue@3/dist/vue.global.js"></script>
<script src="https://cdn.jsdelivr.net/npm/viewlogic/dist/viewlogic-router.umd.js"></script>

<script>
    // Development mode - loads files directly from src/
    ViewLogicRouter({
        environment: 'development',
    });
</script>
 
<script>
    // Production mode - loads pre-built bundles from routes/
    ViewLogicRouter({
        environment: 'production',
        useI18n: true,
        logLevel: 'error'
    });
</script>
